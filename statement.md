# Project Statement – Simple Expense Tracker
# 📝 Problem Statement
Many individuals struggle to monitor their daily expenses effectively.
Lack of tracking often leads to overspending, poor budgeting, and limited financial awareness.
This project aims to provide a simple tool for recording and analyzing personal spending in an easy and visual manner.

# 🎯 Scope of the Project
The Expense Tracker:

Records multiple expenses with categories
Displays a complete list of expenses
Generates a summary report with totals and percentages
Includes a visual pie chart for spending distribution
Helps users understand and improve their spending behavior
Scope is limited to console-based functionality, without databases or file storage.

# 👥 Target Users
Students managing monthly allowances
Working professionals tracking expenses
Anyone wanting a simple non-database expense tracking tool
Beginner programmers learning Python projects
# 🔍 High-Level Features
Add Expense
Input name, amount, and category.

View Expenses
Display all expenses in an organized list.

Expense Report

Total spending
Category-wise breakdown
Percentage distribution
Spending feedback
Pie chart visualization
Interactive Menu System
Easy navigation for users.
